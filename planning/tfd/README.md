# tfd plan solver

## instalation

```
mkdir ~/tfd_planning
cd tfd_planning
**install_tfd**
**clone this repo**

echo alias tfd_launch='[launch_tfd.sh path] $1 $2' >> .bashrc
cd ~/
source .bashrc
```
* **installation**: [official tfd repository](https://github.com/IntelligentRoboticsLabs/plansys2_tfd_plan_solver)


## Launch tfd

```
tfd_launch [domain].pddl [problem].pddl
```
